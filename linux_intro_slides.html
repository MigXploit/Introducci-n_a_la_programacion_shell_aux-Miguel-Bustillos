Concepto base

¿Qué es Linux?

Linux es un sistema operativo de código abierto basado en Unix. Fue creado por Linus Torvalds en 1991 como proyecto universitario.

  • Es el kernel (núcleo) del sistema — el intermediario entre hardware y software
  • Es gratuito y de código abierto — puedes ver, modificar y distribuir el código
  • Altamente estable y seguro — preferido para servidores y sistemas críticos
  • Multitarea y multiusuario — múltiples procesos y usuarios simultáneos

Herramienta principal

La Shell

La shell es el intérprete de comandos — el punto de comunicación entre tú y el sistema operativo.

  • Recibe comandos escritos por el usuario — interfaz de texto (CLI)
  • Los interpreta y los envía al kernel — quien habla con el hardware
  • Devuelve el resultado al usuario — en texto en pantalla
  • Bash es la shell más usada — Bourne Again Shell

Estructura del sistema

Sistema de Archivos

En Linux todo es un archivo. La estructura es un árbol que comienza en la raíz /

🔑 Diferencias clave vs Windows:
• No hay letras de unidad (C:, D:) — todo parte de /
• Las rutas usan / no \
• Mayúsculas y minúsculas son diferentes
• Los archivos ocultos empiezan con punto (.bashrc)
/ ← raíz del sistema
├── home/ ← carpetas de usuarios
│ └── juan/ ← tu directorio (~)
├── bin/ ← comandos esenciales
├── etc/ ← archivos de configuración
├── var/ ← logs y datos variables
├── tmp/ ← archivos temporales
├── usr/ ← programas instalados
├── dev/ ← dispositivos del sistema
└── root/ ← home del superusuario

Concepto fundamental

Permisos en Linux

Cada archivo tiene permisos para 3 grupos: dueño, grupo y otros.

d

w
x

d=dir dueño grupo otros
  • r = read → permiso de lectura (4)
  • w = write → permiso de escritura (2)
  • x = execute → permiso de ejecución (1)
# Ver permisos
$ ls -l script.sh
-rw-r--r-- 1 juan juan 128 Feb 19 script.sh

# Dar permiso de ejecución
$ chmod +x script.sh
$ ls -l script.sh
-rwxr-xr-x 1 juan juan 128 Feb 19 script.sh

# Con números (octal)
$ chmod 755 script.sh
# 7=rwx 5=r-x 5=r-x
Productividad

Atajos y trucos del terminal

⬆️ Historial de comandos

Usa ↑ y ↓ para navegar por comandos anteriores. Ctrl+R para buscar en el historial.

⌨️ Autocompletar con Tab

Presiona Tab para autocompletar nombres de archivos y comandos. Doble Tab muestra todas las opciones.

✂️ Atajos de edición

Ctrl+C cancela el comando actual. Ctrl+L limpia la pantalla. Ctrl+A va al inicio de la línea.

🔗 Encadenamiento de comandos

&& ejecuta el siguiente si el anterior tuvo éxito. | pasa la salida de uno como entrada del siguiente.

📋 Comodines (wildcards)

* representa cualquier cosa: ls *.sh lista todos los scripts. ? representa un solo carácter.

📦 Redirección

> guarda la salida en un archivo. >> agrega al archivo existente. < lee desde un archivo.
